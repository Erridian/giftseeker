# GiftSeeker

*([English](README.md) | Русский)*

> Приложение для автоматического участия в розыгрышах игр на различных сайтах.  
> Программа разработана на Node.js. Интерфейс построен на базе Electron.

## Поддерживаемые сайты

- steamgifts.com
- indiegala.com
- opiumpulses.com
- **mannco.store** Временно выключен

## 🛡 Обновление 2.2.7 (Форк от Erridian)
В этой версии приложения (`2.2.7 By Erridian`) добавлены новые интеграции, значительные улучшения интерфейса и глобальные исправления:


- Устранены ошибки отображения `Disconnected` в сервисе **OpiumPulses**, связанные с потерей кастомных заголовков при редиректах.
- Исправлены падения приложения в **IndieGala** при отсутствии кнопок участия.
- Исправлены проблемы соединения и бесконечные циклы авторизации в **SteamGifts** за счет перевода сетевых запросов маршрутизации на нативный API `electron.net`, что эмулирует реальный браузер и обходит блокировки Cloudflare.
- **Настройка папки сохранения (Защита SSD):** Добавлена возможность изменить папку хранения кэша движка Chromium на жесткий диск, что решает проблему износа SSD из-за огромного потока фонового кэша в `%APPDATA%\GiftSeeker`. Работает из Настроек.
- **Исправление Автозапуска:** Логика «Запускать при старте системы» переписана с забагованной библиотеки `auto-launch` на встроенный системный API Electron `app.setLoginItemSettings()`. Это навсегда решает проблему нерабочих путей в реестре для скомпилированных файлов.
- **Portable версия:** Настроен скрипт сборки Portable версии программы одним файлом, который работает без установки. Умный перехват переменной окружения так же позволяет функции Автозапуска правильно работать даже из временного распаковщика.

## Для разработчиков (Запуск исходного кода)

Вам потребуется скачать [Node.js](https://nodejs.org/) версии **>= 14.15.3**.
Для скачивания приложения через консоль должен быть установлен [Git](https://git-scm.com/install/windows).

1. `git clone https://github.com/Erridian/giftseeker.git`
2. `cd giftseeker`
3. `npm install`
4. `npm run start:ui` или `npm run start:cli`

После этого перед вами откроется десктопная или консольная версия.

## Компиляция (.exe)

Для сборки проекта в исполняемые установщики используется пакет [electron-builder](https://github.com/electron-userland/electron-builder).

**Используйте следующие команды:**

- Сборка стандартного Windows Установщика:
```shell
npm run dist:win
```
- Сборка Portable-приложения одним файлом (без установки):
```shell
npm run dist:portable
```
